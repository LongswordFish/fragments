# fragments

Microservices fragments created for CCP555

## Project Description

This is a microservice API that will run on the AWS.

## How to run

Install the dependencies:

        cd fragments
        npm install

Run eslint:

        npm run lint

Start the server:

        npm run start


Run in development mode:

        npm run dev


Run in debug mode:

        npm run debug
